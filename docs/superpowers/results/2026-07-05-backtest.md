# Prediction backtest results — 2026-07-05

Run: `suncast-backtest` on the Pi against 789 hourly rows (35 days of ERA5 +
Victron), 12 clean days. All metrics leave-one-day-out (out-of-sample).

```text
model                   MAE(bulk,W)   bias(W)  MAE(clean,Wh)   vs M0
--------------------------------------------------------------------
M0 flat                        26.7      -2.6            223      0%
M1 temp(fixed)                 27.0      -5.6            210      1%
M2 temp(fitted)                26.6      -0.9            251     -0%

  M0 flat: k=0.477 gamma=0.0000
  M1 temp(fixed): k=0.477 gamma=-0.0040
  M2 temp(fitted): k=0.444 gamma=0.0065
```

## Verdict: NO-GO on a temperature term — keep the flat factor

- **M2 (fitted) does not win both metrics.** It ties M0 on bulk MAE (−0%) and is
  clearly worse on the clean-day metric (251 vs 223 Wh). The design requires an
  improvement on *both*; it fails.
- **The fitted temperature coefficient is +0.0065 — the wrong sign.** A real
  panel temperature coefficient is ≈ −0.004 /°C (power drops as the cell heats).
  A positive fitted gamma means the fit found no genuine temperature signal in
  the bulk-hour data; it is fitting noise.
- **M1 (fixed literature coefficient) is slightly worse on bulk** (+1%), better
  on clean-day (210) — inconsistent, no clear win.

## Why the bright-day ratio correlation did not become a temperature win

The motivating observation (bright days showed lower forecast-vs-actual ratios)
was measured over whole-day bulk hours. On bright days the battery fills fast, so
the surviving bulk hours are low-sun morning shoulders — a **sampling bias**, not
a temperature effect. Scored per bulk hour out-of-sample, no temperature signal
survives.

## Decision

Keep the KISS flat calibration factor (live value 0.473; this backtest
independently recovers k = 0.477 — consistent). Do **not** wire a temperature
model into the live predictor. The remaining ~27 W bulk MAE (~20% of a typical
bulk-hour output) is irreducible forecast + conversion error that temperature
does not explain.

Revisit only if a materially different candidate appears (e.g. sun-elevation /
air-mass term), and only via this same out-of-sample harness.
